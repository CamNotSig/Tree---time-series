# Árbol de Series de Tiempo

Aplicación interactiva construida en React que permite explorar de forma visual la clasificación de modelos de series temporales en econometría.

## Características

- Navegación tipo árbol
- Clasificación de modelos:
  - Estacionarios
  - No estacionarios
  - Modelos no lineales
  - Machine Learning

## Tecnologías

- React
- Vite
